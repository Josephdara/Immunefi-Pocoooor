## Foundry

Foundry consists of:

-   **Forge** 
-   **Cast** 
-   **Anvil** 
-   **Chisel** 

## Documentation

For any foundry related issue checkout the docs at: https://book.getfoundry.sh/

## Usage

### Update foundry
```shell
$ foundryup
```

### Fork with anvil
```shell
$ anvil --fork-url https:infuria-api
```

### Test
```shell
$ forge test
```

### Help

```shell
$ forge --help
$ anvil --help
$ cast --help
```
